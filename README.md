🛒 Inventory Management System

📌 Project Overview
* The Inventory Management System is a basic Python application designed to manage product categories and product lists in a retail shop. It demonstrates how fixed data (categories) and dynamic data (products) can be handled using Python data structures.
* This project focuses on fundamental concepts like tuples, lists, user input, and data manipulation

🎯 Objective
* To manage fixed product categories
* To allow dynamic addition and removal of products
To demonstrate the use of tuples and lists in real-world scenarios

⚙️ Tools & Technologies Used
& Python – Core programming language
& Google Colab

🛠️ What I Have Done
& Created a tuple to store fixed product categories (immutable data)
& Created a collection of products to represent inventory items
* Implemented functionality to:
    Accept user input to add new products
    Dynamically update the product list
* Allowed users to:
    Enter an index value
    Remove a product from the list
* Displayed:
    Updated product list after modifications
    Fixed product categories

💡 Key Features
* Use of tuple for fixed categories (cannot be modified)
* Use of list/tuple operations for dynamic updates
* Interactive user input handling
* Basic inventory update functionality
* Beginner-friendly implementation

⚠️ Note
* Tuples are immutable, so adding new items creates a new tuple
* For deletion operations, lists are more suitable than tuples

📚 Conclusion

* This project provides a basic understanding of inventory handling using Python data structures. It highlights the difference between mutable and immutable data types and demonstrates simple CRUD-like operations (Create, Read, Update, Delete).
